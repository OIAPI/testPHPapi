# 幻影坦克
-----
### 介绍
幻影坦克是一种图片显示方式，它会在黑色背景下显示一张图片，白色背景下显示另一张图片，从而形成一种幻影坦克的效果。

### 提示
该项目由 DeepSeekv3 模型进行 py -> php，可能存在一定的错误。

### 意外的情况
由于php-fpm的特性，该项目可能会吃较大的内存。
如果内存占用过多，可以尝试关闭php-fpm的opcache功能。
